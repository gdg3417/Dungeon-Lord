What is the smallest version of Dungeon Builder that proves the fantasy is fun?

**MVP gameplay scope**

Include

- Dungeon core

- 1 main dungeon with up to 5 floors

- 2 monster families only (Undead and Goblinoid)

- Mana generation

- Basic heat system with only Peace, Notice, Concern

- Adventurers with 5 classes only

- Loot tiers up to Steel

- Research system with 1 slot

- Idle offline mana calculation

- One sub dungeon type (Mana Farm)

Explicitly exclude for MVP

- Prestige

- Seasonal events

- PvP or leaderboards

- Hero adventurers

- More than 1 sub dungeon

- Advanced diplomacy

- More than 3 heat states

- Boss sets beyond one per family
